**Idle state (no conversation selected):**
- Same centered welcome as current: "DocuMind" heading, "What would you like to know?", suggestions
- Input is centered too
- On first message: transitions to chat layout

**Active state (conversation going):**
- Messages scroll up as new ones arrive
- Input fixed at bottom
- Auto-scroll to latest message
- Streaming: assistant message grows in real-time with cursor

**State management:**
- currentConversationId: string | null
- messages: Message[] (loaded from API or accumulated during session)
- streaming: boolean
- streamingContent: string (current assistant message being streamed)

**API integration:**
- New chat: POST /api/chat → get session ID, start streaming
- Follow-up: POST /api/chat/[id] → stream response, add to messages
- Load conversation: GET /api/chat/[id]/messages → populate messages array
